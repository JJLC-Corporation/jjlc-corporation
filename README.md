
# Welcome to .Lift()!

## Running this locally

<ol>
  <li>Clone the repository</li>
  <li>cd into the application</li>
  <li>$ yarn</li>
  <li>$ bundle</li>
  <li>$ rails db:setup</li>
  <li>$ yarn add react-router-dom</li>
</ol>
